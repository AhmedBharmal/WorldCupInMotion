## About the Project

WorldCupInMotion is a frontend web project created around the FIFA World Cup 2026.

The goal of the project is to present football information through a cinematic and interactive interface instead of a normal static website.

The platform focuses on the 2026 World Cup hosted by Canada, Mexico, and the United States.

## Main Features

* Interactive 3D world globe
* Country and national team explorer
* Team squad and player information
* Tournament groups
* Knockout bracket
* Stadium information
* World Cup history
* Tournament statistics
* Player leaderboards
* Match ticker
* Football news section
* Search for teams, players, countries, and stadiums
* Responsive design for desktop and mobile devices
* Animated World Cup timeline and visual effects

## Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/AhmedBharmal/WorldCupInMotion.git
```

Open the project folder:

```bash
cd WorldCupInMotion
```

Start the local server:

```bash
node dev-server.js
```

You can also open the project using the Live Server extension in Visual Studio Code.
